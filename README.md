# Exam Revision Planner
A premium, glassmorphism-based exam revision planner designed to help students organize their study sessions and track progress effectively.

## Features
- **Smart Planning**: Automatically distributes subjects and topics based on your exam date and daily study hours.
- **Glassmorphism UI**: A beautiful, modern interface with fluid animations and a premium feel.
- **Focus Timer**: Integrated Pomodoro-style timer to track focus sessions.
- **Analytics**: Visual distribution of study time across different subjects.
- **Smart Review**: Automatic scheduling of review sessions based on topic difficulty.

## Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS with Glassmorphism utilities, Tailwind CSS
- **State**: Custom hooks with LocalStorage persistence

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
